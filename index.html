About This Project

Project Overview

This interactive web application is the culmination of a comprehensive data science project based on the famous "Titanic: Machine Learning from Disaster" challenge from Kaggle. The goal was to analyze historical passenger data to understand which factors most significantly contributed to survival.

Through rigorous data analysis and machine learning, we've built a predictive model that can estimate survival probability based on passenger characteristics, providing insights into one of history's most tragic maritime disasters.

Technical Details

The predictor uses a Logistic Regression model trained on the original Titanic dataset. Key features include passenger class, sex, age, fare, port of embarkation, and family size (siblings/spouses and parents/children).
