About This Project

Learn about the technology, data, and team behind this interactive Titanic survival predictor

Project Overview

This interactive web application is the culmination of a comprehensive data science project based on the famous "Titanic: Machine Learning from Disaster" challenge from Kaggle. The goal was to analyze historical passenger data to understand which factors most significantly contributed to survival.

Through rigorous data analysis and machine learning, we've built a predictive model that can estimate survival probability based on passenger characteristics, providing insights into one of history's most tragic maritime disasters.

The application combines modern web technologies with data science to create an educational and engaging experience that brings historical data to life through interactive visualizations and real-time predictions.

Technical Details

The predictor uses a Logistic Regression model trained on the original Titanic dataset. Key features include passenger class, sex, age, fare, port of embarkation, and family size (siblings/spouses and parents/children).

Model Performance

~76.5% Accuracy

Demonstrating that survival was far from random and heavily influenced by these factors

Data Sources & Methodology

Dataset Features

  • ✓ Pclass: Passenger class (1 = 1st, 2 = 2nd, 3 = 3rd)
  • ✓ Sex: Gender (0 = Female, 1 = Male)
  • ✓ Age: Age in years
  • ✓ SibSp: Number of siblings/spouses aboard
  • ✓ Parch: Number of parents/children aboard
  • ✓ Fare: Passenger fare in British pounds
  • ✓ Embarked: Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton)

Data Processing

Our data preprocessing pipeline includes several critical steps to ensure accurate predictions:

  • • Missing Value Imputation: Strategic filling of missing age and fare data using median values
  • • Feature Engineering: Creation of family size and other derived features
  • • Categorical Encoding: One-hot encoding for categorical variables
  • • Feature Scaling: Standardization of numerical features
  • • Cross-validation: K-fold validation for robust model evaluation

Historical Context

The RMS Titanic, often called the "unsinkable ship," was the largest passenger steamship in the world when it set sail on its maiden voyage on April 10, 1912. Built by the White Star Line, it was a marvel of engineering luxury, designed to transport passengers between Southampton, England, and New York City.

Tragically, just four days into its journey, on April 15, 1912, the Titanic struck an iceberg and sank in the North Atlantic Ocean, resulting in one of the deadliest maritime disasters in history. The disaster led to major improvements in maritime safety regulations and procedures.

Key Facts

  • • Length: 882 feet 9 inches
  • • Height: 175 feet (from keel to funnels)
  • • Capacity: 2,435 passengers, 892 crew
  • • Lifeboats: Only 20 (capacity for 1,178 people)
  • • Survivors: 710 out of 2,224 aboard

"Women and Children First"

The famous maritime protocol was followed during the evacuation, which explains why women and children had significantly higher survival rates. This social norm, combined with class-based access to lifeboats, created the survival patterns we analyze today.
